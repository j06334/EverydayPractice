## 题目：给你一个字符串 `s`，找到 `s` 中最长的回文子串。

示例：

```
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。

输入：s = "cbbd"
输出："bb"

输入：s = "a"
输出："a"

输入：s = "ac"
输出："a"

提示：

1 <= s.length <= 1000
s 仅由数字和英文字母（大写和/或小写）组成
```

tag: 字符串，动态规划

## 题解：

动态规划：dp{i}{j}表示字符串S的下标i到j之间是否回文

dp【i】【j】= (nums[i]==nums[j]) && (dp【i】【j】)

判断回文的思路：

1. 先判断S(i~j)字符串的首尾两个字符是否相等，如果不相等则S(i~j)不是回文字符串；

2. 如果首尾两个字符相等，则去除这两个字符，形成新的子串(由子串是否回文决定)，再重复步骤1



```c++
class Solution {
public:
    string longestPalindrome(string s) {
        string res="";
        int resLen=1;
        int resStartIdx=0;
        
        int len=s.length();
        if(len<2)
        {
            return s;
        }

        int **dp = new int* [len];
        for(int i=0; i<len; i++)
        {
            dp[i] = new int [len];
            dp[i][i] = 1;  //初始化
        }

        for(int j=1; j<len; j++)
        {
            for(int i=0; i<j; i++)
            {
                if(s[i] != s[j])
                {
                    dp[i][j]=0;
                }
                else if(j-i<3) //当s[i] == s[j]，3个或2个字母判断是否回文
                {
                    dp[i][j] = 1;
                }
                else  //当s[i] == s[j]，字母间隔大于3，判断是否回文，这就要将首尾去掉，再判断中间子串是否回文
                {
                    dp[i][j] = dp[i+1][j-1];
                }
                
                if(dp[i][j]==1)
                {
                    if(j-i+1 > resLen)
                    {
                        resLen=j-i+1;   //更新记录最大的回文长度，并且记录回文开始的下标
                        resStartIdx=i;
                    }
                }
            }
        }
        //return s.substring(resStartIdx, resStartIdx+resLen); //包含前idx，不包含后idx的内容
        for(int i=resStartIdx; i<resStartIdx+resLen; i++)   //组装返回值
        {
            res += s[i];
        }
        return res;

    }
};
```
